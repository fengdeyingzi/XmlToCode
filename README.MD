# XmlToCode

将安卓xml转换成代码，此项目仅为测试项目，还不够完善，有什么建议或意见欢迎提出。

> 注：本项目仅支持部分xml属性及安卓控件转换

目前已初步实现xml布局转安卓java/swift/flutter代码
